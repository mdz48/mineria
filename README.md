# Banco de Trabajos de Minería de Datos y Proyecto Integrador

Este repositorio actúa como un banco centralizado para todos los trabajos prácticos, análisis y módulos desarrollados en la asignatura de Minería de Datos, además de alojar la fase analítica y de entrenamiento para el Proyecto Integrador.

## Evaluaciones y Secciones

- **EVA02**: Para acceder a los contenidos de la evaluación EVA02, dirígete a la carpeta `predictivo/`.
  - Contiene los notebooks de modelado predictivo, procesamiento de datos y la documentación analítica asociada.

## Estructura del Repositorio

- `predictivo/`: Código, datos y desarrollo correspondiente a la EVA02 y modelos predictivos.
- `clustering/`: Algoritmos, modelos y evaluaciones de agrupamiento (K-Means, Fuzzy C-Means, etc.).
- `time series/`: Procesamiento y análisis de series temporales.
- `reportes_latex/`: Informes técnicos y documentación académica en LaTeX.

## Entorno Virtual

Todas las dependencias del proyecto se gestionan de manera local dentro del entorno virtual `.venv`.
